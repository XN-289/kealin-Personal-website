# sac-ai.com 深度考察报告

## 一、项目概况

- **仓库**: github.com/Sac-Y/sac-ai.com
- **定位**: Sac 的个人网站，展示 AI 内容实践、视频创作和 build in public
- **技术栈**: 纯 HTML + CSS + JS（无框架），GSAP 动画库
- **部署**: Cloudflare Pages

## 二、设计系统

### 2.1 色彩

| 变量 | 值 | 用途 |
|:---|:---|:---|
| `--paper` | `#F4EFE6` | 纸张底色（暖米色） |
| `--ink` | `#1A1714` | 主文字（深墨色） |
| `--accent` | `#F0652E` | 强调色（橙色，像印章） |
| `--line` | `#C9C0B2` | 线条、边框 |
| `--muted` | `#7d756a` | 次要文字 |
| `--ink-soft` | `#3a342d` | 柔和文字 |

**核心理念**: 纸张 + 墨水，像在纸上写字。橙色是"印章"，点缀而非主色。

### 2.2 字体

| 变量 | 值 | 用途 |
|:---|:---|:---|
| `--serif` | Playfair Display + Noto Serif SC | 标题、大字 |
| `--serif-cjk` | Noto Serif SC + Playfair Display | 中文标题优先 |
| `--sans` | Inter + Noto Sans SC | 正文 |
| `--sans-cjk` | Noto Sans SC + Inter | 中文正文优先 |
| `--mono` | SF Mono / Menlo | 标注、代码 |

**关键**: 标题用衬线体（Playfair Display），正文用无衬线体（Inter）。中文用思源宋体。

### 2.3 间距

- `--nav-h: 78px` — 导航高度
- `--pad-x: clamp(32px, 5vw, 72px)` — 水平内边距（响应式）
- `--rail-x: 48px` — 左侧脊柱线位置

### 2.4 动画

- 缓动: `cubic-bezier(.22, .61, .36, 1)` — 先快后慢，自然感
- 入场: opacity 0→1, y 20→0, duration 1s, stagger 0.14s
- 滚动触发: start "top 76%"

### 2.5 纹理

- **点阵纹理**: `radial-gradient(rgba(26,23,20,.02) 1px, transparent 1px)`, 3px 间距
- **噪点纹理**: SVG feTurbulence 滤镜，opacity 0.045，multiply 混合模式

## 四、交互系统

### 4.1 GSAP 入场动画

```javascript
// Hero 入场
gsap.set(heroItems, { opacity: 0, y: 20 });
gsap.to(heroItems, {
  opacity: 1, y: 0, duration: 1, ease: "power3.out",
  stagger: 0.14, delay: 0.25,
});

// 后续区块
gsap.set(items, { opacity: 0, y: 30 });
gsap.to(items, {
  opacity: 1, y: 0, duration: 1.4, ease: "power3.out", stagger: 0.2,
  scrollTrigger: { trigger: section, start: "top 76%" },
});
```

### 4.2 导航高亮

- IntersectionObserver 监听每个 section
- rootMargin: "-45% 0px -45% 0px"（只有在视口中间才触发）
- 切换 `.is-active` class

### 4.3 粒子系统

- 从头像图片采样像素
- 非背景像素转为粒子（墨色 + 橙色）
- 入场: 粒子从左侧聚合到目标位置（1.6s）
- 成形后: 轻微呼吸动画
- hover: 粒子被鼠标推开（半径 115-180px）
- 粒子数量: 移动端 11000，桌面端 30000

### 4.4 终端打字

- 8 行代码，7.2s 循环
- 滚动触发播放，离开暂停
- 进度条 + 状态文字同步更新

### 4.5 平滑滚动

- 锚点链接 `#xxx` 点击时 `scrollIntoView({ behavior: "smooth" })`

## 六、技术要点

1. **无框架**: 纯 HTML/CSS/JS，性能极好
2. **GSAP**: 所有动画用 GSAP，包括 ScrollTrigger
3. **SVG 滤镜**: 双色调效果（duotone）
4. **Canvas**: 粒子系统
5. **CSS 变量**: 统一设计 token
6. **BEM 命名**: `.block__element--modifier`
7. **data-reveal**: 统一的入场动画标记
8. **prefers-reduced-motion**: 无障碍支持
